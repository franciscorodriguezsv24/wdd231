Attributions

This page lists the resources and references used to build the SkillConnect SV website for the BYU-Idaho WDD 231 final project.

Concept & content

  • SkillConnect SV is an original concept created for this course. It is a fictional freelance services marketplace.
  • All provider profiles, names, ratings, reviews, biographies, and contact details in data/providers.json are fictional and were written by the author for demonstration purposes. The example email and phone values are not real.

Fonts

  • Poppins typeface, designed by Indian Type Foundry / Jonny Pinhorn, served via Google Fonts (Open Font License).
  • Inter typeface, designed by Rasmus Andersson, served via Google Fonts (Open Font License).

Code

  • modern-normalize v2.0.0 by Sindre Sorhus (MIT License) — used as the CSS reset (styles/normalize.css).
  • All other HTML, CSS, and JavaScript was written by Alejandro Rodriguez for this project.

Images & icons

  • The logo, favicon, hero illustration, category icons, and provider avatars are original SVG graphics created by the author for this project.
  • Emoji used in the interface are standard Unicode characters rendered by the user's device.

Reference

  • MDN Web Docs — reference for the Fetch API, the native <dialog> element, and form validation.
